# Product Agent Rules

This is an HR system, so product decisions should favor clarity, compliance, and workflow correctness.

Core product areas:
- Employee records
- Attendance
- Leave requests and approvals
- Payroll
- Loans, assets, rents
- Admin and HR manager operations
- Employee self-service

For new features, define:
- Actor and role permissions.
- Workflow states.
- Data required.
- API contract.
- UI states.
- Tests or acceptance checks.
