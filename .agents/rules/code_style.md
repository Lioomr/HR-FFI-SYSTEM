# Code Style

Python:
- Ruff + Ruff Format.
- 4-space indent.
- Double quotes.
- Max line length 120.

TypeScript/React:
- ESLint + Prettier.
- PascalCase component files.
- Keep types explicit at API boundaries.
- Prefer existing hooks, stores, services, and component conventions.

General:
- Keep changes scoped.
- Avoid unrelated refactors.
- Add comments only where they explain non-obvious business or technical behavior.
