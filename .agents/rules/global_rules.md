# Global Rules

Always obey root `AGENTS.md`.

Project principles:
- Plans are source contracts, especially `plans/Global API Rules (v1).txt`.
- Preserve existing public behavior unless the task explicitly changes it.
- Validate role-based access on the backend.
- Keep secrets out of the repository.
- Prefer small, reviewable changes.

When uncertain:
- Inspect existing implementation.
- Search plans and docs.
- State the assumption in the final response if it affects behavior.
