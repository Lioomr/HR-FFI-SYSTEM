# Frontend UI Direction Skill

Use for UI/UX implementation and visual cleanup.

Direction:
- HR admin screens should be calm, structured, and information-dense.
- Use Ant Design components consistently.
- Use icons for common actions where existing UI does.
- Preserve clear hierarchy: page title, actions, filters, content, details.
- Avoid decorative UI that competes with operational data.

Always check responsive behavior for tables, forms, and action bars.
