# Frontend MVP Tasks

Use this as a lightweight checklist for new frontend feature screens:
- Confirm route and navigation entry.
- Define API service and TypeScript types.
- Build list/detail/form states as needed.
- Add loading, empty, error, and permission states.
- Wire create/update/delete or workflow actions.
- Add behavior-critical tests where practical.
- Run type-check, lint, and build when relevant.
