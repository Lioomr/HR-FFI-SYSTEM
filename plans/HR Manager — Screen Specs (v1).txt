HR Manager — Screen Specs (v1)
1) Login

Purpose
Allow HR Manager to access the system securely.

Components

Email

Password

Login button

Actions

Submit login

Validations

Email format

Required fields

Rate-limit after repeated failures

Audit logs

Login success / login failure (include IP)

API needs

POST /auth/login

2) Change Password

Purpose
HR Manager can change their own password.

Components

Current password

New password

Confirm new password

Actions

Save changes

Validations

New password strength policy

New password ≠ old password

Confirm password match

Audit logs

Password change event

API needs

POST /auth/change-password
